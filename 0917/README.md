### 퓁 폼

- 웹 페이지에서 사용자 입력을 받는 폼
- 로그인, 등록, 검색, 예약 등

### 폼 작성

name 속성
- 폼의 이름 지정

action 속성
-  폼 데이더를 처리할 웹 서버 응용 프로그램
-  submit 버튼이 눌리면 브라우저는 action에 지정된 웹 서버 응용프로 그램 실행 요청
- Java, JSP, C/C++ 등으로 작성

method 속성
- 폼 데이터를 웹 서버로 전송하는 형식

폼은 사용자에게 무언갈 입력받고 싶을 때 쓰는 것
